# MechPro Experts

Premium B2B2C automotive service, claim management and lead management platform.

## Monorepo Structure

- frontend/ -> Next.js + TypeScript + Tailwind website and portal UI
- backend/ -> FastAPI services (upcoming phases)
- database/ -> PostgreSQL schema and migration assets (upcoming phases)
- docs/ -> architecture and implementation planning
- uploads/ -> local upload staging (dev only)
- services/ -> shared service contracts and integration notes

## Frontend (Phase 1)

Implemented pages:
- /
- /about
- /services
- /contact
- /login

## Run Frontend

```bash
cd frontend
npm install
npm run dev
```
